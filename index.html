스타트업 파트너센터

초기 스타트업의 성장 단계를 진단하고, 검증된 전문가를 매칭하여 필요한 실행 과제를 원스톱으로 해결하는 프리미엄 B2B 컨시어지 서비스

서비스 소개

스타트업 파트너센터는 플랫폼이 아닌 "딜 메이커"를 지향합니다. 운영자가 직접 전문가를 수동 매칭하고, 가격을 재구성하여 2개 안을 제시하는 프리미엄 B2B 컨시어지 모델입니다.

핵심 정책

절대 변경해서는 안 되는 프로젝트 핵심 원칙

⚠️ 주의
아래 정책은 서비스의 근간을 이루는 원칙으로, 어떤 상황에서도 변경하지 않습니다.

1. 플랫폼 이용 완전 무료

스타트업이 서비스를 부담 없이 이용할 수 있도록 플랫폼 자체 이용료는 0원입니다. 수익은 아래 세 가지 채널로만 발생합니다.

  • 문서 템플릿 판매 (건당 49,000원~190,000원)
  • 전문가 업무 결제 수수료 (15%)
  • 매칭 마진 (20~30%)

2. 소셜 로그인

Google + 이메일만 지원합니다. 네이버/카카오 로그인은 사용하지 않습니다.

3. 파트너 개념 없음

서비스에 "파트너" 사용자 유형은 존재하지 않습니다. 이용자만 존재하며, 전문가는 관리자 영역에서 별도 관리합니다.

💡 참고
"운영 파트너", "문서 파트너" 등의 표현은 서비스 카테고리 이름이지, 사용자 유형이 아닙니다.

4. 명칭 규칙

표기실제 의미비고
운영 파트너전문가 매칭 서비스 카테고리세무/회계/법률/노무
문서 파트너문서 마켓 서비스 카테고리템플릿/맞춤 의뢰
스타트업피드커뮤니티 기능URL은 /community 유지

5. 통화 표시

모든 금액은 한국 '원'으로 표시합니다. 달러($) 기호는 절대 사용하지 않습니다.

문서 파트너

문서 마켓 — 템플릿 판매 및 맞춤 의뢰 시스템 (주요 수익원)

기능 개요

문서 파트너는 사업계획서, 제안서, IR Deck, 회사소개서 등의 전문 문서 템플릿을 판매하고, 맞춤 작성 의뢰를 연결하는 서비스입니다. 주요 수익원으로서 퍼블릭 페이지에서도 비회원에게 노출되어 가입을 유도합니다.

주요 기능

  • 템플릿 판매 — 건당 49,000원~190,000원의 전문 문서 템플릿
  • 맞춤 의뢰 — 스타트업 맞춤 문서 작성 전문가 연결
  • 자동 추천 — 공고 매칭 태그 기반 관련 문서 자동 추천
  • 퍼블릭 노출 — 비회원도 마켓을 둘러볼 수 있음 (장바구니 시 로그인 유도)

관리자 상품관리

관리자 센터의 문서상품관리는 imweb 스타일로 구현되어 있습니다. 상품 등록/수정/삭제, 가격 설정, 태그 관리, 공개/비공개 토글 등을 지원합니다.

AI 투자

VC/엔젤 투자사 매칭, 투자 준비도 진단, IR 가이드

기능 개요

AI 투자는 스타트업의 투자 유치를 지원하는 모듈입니다. 투자 준비도를 진단하고, 적합한 투자사를 매칭하며, IR 자료 준비를 가이드합니다.

주요 기능

  • 투자 준비도 진단 — 스타트업의 현재 투자 준비 상태를 체계적으로 평가
  • VC/엔젤 투자사 매칭 — 산업, 규모, 라운드에 맞는 투자사 자동 추천
  • IR 가이드 — IR Deck 작성 가이드 및 발표 코칭 연결

채팅 시스템

Supabase Realtime 기반 실시간 상담 + 업무 결제 통합 채팅

기능 개요

채팅 시스템은 Supabase Realtime을 활용한 실시간 메시징 시스템입니다. 단순 텍스트 채팅을 넘어, 파일 전송과 업무 결제 요청까지 처리하는 통합 커뮤니케이션 채널입니다.

메시지 타입

message_type설명예시
text일반 텍스트 메시지상담 대화
file파일 첨부문서, 이미지 전송
payment_request업무 결제 요청파트너가 의뢰자에게 요청
payment_completed결제 완료 알림결제 확인 메시지
system시스템 메시지입장/퇴장, 상태 변경

삭제 정책

💡 소프트 삭제 방식
채팅방은 hidden_by TEXT[] 필드로 소프트 삭제됩니다. 사용자가 채팅방을 삭제하면 해당 사용자의 ID가 hidden_by 배열에 추가되어 목록에서 숨겨집니다. 상대방이 새 메시지를 보내면 자동 복원됩니다.

알림 시스템

notifications 테이블에 알림을 저장하고, 헤더의 벨 아이콘에 읽지 않은 알림 수를 배지로 표시합니다.

관리자 대시보드는 권한별 개인화되어 표시됩니다. 각 관리자 역할에 따라 표시되는 위젯과 데이터가 다릅니다.

주요 관리 기능

  • 회원관리 — 전체/활성/삭제 탭 분리, 회원 검색/필터
  • 전문가관리 — 전문가 정보 수정 + 비밀번호 변경
  • 문서상품관리 — imweb 스타일 상품 등록/수정/삭제
  • 공고관리 — 외부 공고 수집 버튼, 공개/비공개 토글
  • 피드관리 — RSS 수집, 큐레이션, 공개/비공개
  • 채팅관리 — 채팅방 목록 조회, 삭제
  • 정산관리 — 정산 현황 조회, 상태 변경

3차원 태그

공고와 문서는 사업유형, 문서용도, 대상단계의 3가지 축으로 태깅됩니다.

사업유형 (10개)

사업화 기술개발(R&D) 시설·공간·보육 멘토링 인력 판로·해외진출 융자·보증 행사·네트워크 정책자금 투자

문서용도 (9개)

사업계획서 기술제안서 IR Deck 회사소개서 연구개발계획서 입주신청서 수출계획서 자금조달계획서 제안서(일반)

대상단계 (4개)

예비창업자 초기창업(1~3년) 성장기(3~7년) 스케일업(7년 이상)
